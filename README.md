# ॐ Jogi Baba — Sacred Soles
**"Where the Sacred Meets the Street"**

A high-concept, luxury e-commerce landing page for a performance footwear brand. This project explores the intersection of ancient Indian sacred geometry and modern athletic technology.

[Live Demo](https://joblesskid-2510.github.io/jogi-baba-sacred-soles/)

---

## 🎨 Aesthetic & Vision
The design utilizes a **Saffron & Indigo** palette—traditional colors of devotion and the infinite—blended with a dark "Charcoal" UI to create a premium, nocturnal feel. 

### Key Features:
* **Sacred Cursor:** A custom reactive cursor system with a trailing ring that grows on interactive elements.
* **Procedural Background:** A dynamic HTML5 Canvas background featuring floating 'Om' particles and sacred connection lines.
* **SVG Artistry:** Custom-drawn SVG footwear and mandalas that animate on scroll and hover.
* **Responsive Narrative:** A fully fluid layout that adapts from wide desktop "Sacred Drops" to mobile-friendly "Sangha" views.

## 🛠️ Technical Zen
* **Architecture:** Semantic HTML5 & CSS3 Power.
* **Typography:** A triad of *Cinzel Decorative* (Heritage), *Cormorant Garamond* (Grace), and *Teko* (Performance).
* **Performance:** Zero external heavy libraries. Vanilla JavaScript handles the scroll reveals, canvas particles, and UI interactions for maximum speed.
* **Micro-interactions:** Custom-built "Add to Cart" toast notifications and color-swatch background tinting.

## 🧘 Architecture of the Soul
* `index.html` - The core structure containing the SVGs and layout.
* `style` - Custom CSS Variables for the "Sacred" color palette.
* `script` - Particle systems, cursor logic, and intersection observers for scroll reveals.
